Ventas

← Volver a inicio

Objetivo

Registrar y controlar las ventas de servicios y refacciones, utilizando un sistema de folios estandarizados, para medir ingresos y relacionarlos con clientes, personal y métodos de pago dentro del taller.

Usuarios del subsistema

  • Cliente
  • Personal administrativo
  • Administrador

Requerimientos por usuario

Requerimientos del Cliente

  • Consultar historial de compras.
    Campos: ID Cliente, ID Venta, Fecha de Venta, Total, Descripción, Estatus
  • Recibir confirmación de ventas realizadas.
    Campos: ID Cliente, ID Venta, Total, Fecha de Venta, Estatus

Requerimientos del Personal Administrativo

  • Registrar ventas.
    Campos: ID Venta, ID Cliente, Fecha de Venta, Descripción, Total, Método de Pago, Estatus, Notas
  • Actualizar información de ventas.
    Campos: ID Venta + cualquier campo editable (Cliente, Fecha, Descripción, Total, Método de Pago, Estatus, Notas)
  • Generar reportes y recibos.
    Campos: ID Venta, ID Cliente, Fecha de Venta, Descripción, Total, Método de Pago, Estatus

Requerimientos del Administrador

  • Registrar, actualizar y borrar ventas.
    Campos: ID Venta, ID Cliente, Fecha de Venta, Descripción, Total, Método de Pago, Estatus, Notas
  • Acceder a información completa del subsistema.
    Campos: Todos los campos (Ventas, Clientes, Métodos de Pago, Estatus, Fechas, Notas)
