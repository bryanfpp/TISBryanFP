Tabla Órdenes de Servicio

Propósito

La tabla Órdenes de Servicio es esencial para registrar, organizar y dar seguimiento a las reparaciones realizadas en el taller. Permite gestionar de manera controlada el ciclo de vida de una orden mediante estados, prioridades y responsables, facilitando la trazabilidad de cada servicio solicitado por los clientes.

Campos

Campo Tipo de dato Descripción
Id_orden_servicioVARCHAR(15) PRIMARY KEYIdentificador único (ej. OS-2025-0001)
Id_clienteINTRelación con la tabla Clientes
Id_equipoINTReferencia al equipo recibido
fecha_recepcionDATETIMEFecha en que se recibió el equipo
accesorios_recibidosNVARCHAR(255)Lista de accesorios entregados junto con el equipo
estado_fisico_inicialNVARCHAR(255)Descripción del estado físico al momento de la recepción
prioridadNVARCHAR(10)Baja, Media o Alta
estatusNVARCHAR(30)Recibida, Espera refacción, En reparación, Lista, Entregada, Cancelada
fecha_estimada_entregaDATETIMEFecha de entrega prevista
Id_diagnosticoINTRelación con la tabla Diagnóstico
garantia_aplicaBITIndica si aplica garantía
Id_garantiaINT NULLRelación con la garantía si aplica
anticipoDECIMAL(10,2)Monto anticipado recibido, 50% OBLIGATORIO
saldoDECIMAL(10,2)Monto pendiente de pago
id_personalINTEmpleado responsable de la orden
notasNVARCHAR(500)Observaciones adicionales

Código SQL (MySQL)

CREATE TABLE Ordenes_Servicio (
  Id_orden_servicio VARCHAR(15) PRIMARY KEY,
  Id_cliente INT NOT NULL,
  Id_equipo INT NOT NULL,
  fecha_recepcion DATETIME NOT NULL,
  accesorios_recibidos NVARCHAR(255),
  estado_fisico_inicial NVARCHAR(255),
  prioridad NVARCHAR(10),
  estatus NVARCHAR(30),
  fecha_estimada_entrega DATETIME,
  Id_diagnostico INT,
  garantia_aplica BIT,
  Id_garantia INT NULL,
  anticipo DECIMAL(10,2),
  saldo DECIMAL(10,2),
  id_personal INT,
  notas NVARCHAR(500)
);
