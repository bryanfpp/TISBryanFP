Tabla Pagos

Propósito

La tabla Pagos permite registrar, controlar y dar seguimiento a todos los pagos realizados por los clientes, incluyendo método de pago, subtotal, descuentos y estatus de la transacción. Facilita la gestión financiera y la relación con órdenes de servicio.

Campos

Campo Tipo de dato Descripción
Id_pagoVARCHAR(15) PRIMARY KEYIdentificador único (ej. PAY-2025-0001)
Id_orden_servicioVARCHAR(15)Referencia a la orden de servicio asociada
Id_clienteINTRelación con la tabla Clientes
fecha_pagoDATETIMEFecha en que se realizó el pago
metodo_pagoNVARCHAR(20)Efectivo, Tarjeta, Transferencia o Mixto
subtotalDECIMAL(10,2)Monto antes de descuentos
descuentoDECIMAL(10,2)Monto descontado
totalDECIMAL(10,2)Total a pagar (calculado)
estatus_pagoNVARCHAR(20)Pendiente, Pagado o Cancelado
id_personalINTEmpleado que registró o procesó el pago

Código SQL (MySQL)

CREATE TABLE Pagos (
  Id_pago VARCHAR(15) PRIMARY KEY,
  Id_orden_servicio VARCHAR(15) NOT NULL,
  Id_cliente INT NOT NULL,
  fecha_pago DATETIME NOT NULL,
  metodo_pago NVARCHAR(20),
  subtotal DECIMAL(10,2),
  descuento DECIMAL(10,2),
  total DECIMAL(10,2),
  estatus_pago NVARCHAR(20),
  id_personal INT
);
