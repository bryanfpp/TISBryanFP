Tabla Ventas

Propósito

La tabla Ventas registra todas las transacciones de servicios y refacciones del taller, permitiendo asociarlas con clientes, personal, pagos y diagnósticos. Facilita el control de ingresos y la generación de reportes de ventas.

Campos

Campo Tipo de dato Descripción
Id_ventaVARCHAR(15) PRIMARY KEYIdentificador único de la venta
fecha_ventaDATETIMEFecha de realización de la venta
id_clienteINTReferencia al cliente
id_personalINTEmpleado que registró la venta
descripcion_ventaNVARCHAR(255)Descripción del servicio o refacción vendida
cantidadINTCantidad de ítems vendidos
totalDECIMAL(10,2)Monto total de la venta
id_pagoVARCHAR(15)Referencia al pago asociado
id_diagnosticoINTReferencia al diagnóstico si aplica
tipoNVARCHAR(20)Tipo de venta: servicio o refacción

Código SQL (MySQL)

CREATE TABLE Ventas (
  Id_venta VARCHAR(15) PRIMARY KEY,
  fecha_venta DATETIME NOT NULL,
  id_cliente INT NOT NULL,
  id_personal INT,
  descripcion_venta NVARCHAR(255),
  cantidad INT,
  total DECIMAL(10,2),
  id_pago VARCHAR(15),
  id_diagnostico INT,
  tipo NVARCHAR(20)
);
