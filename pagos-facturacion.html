Pagos & Facturación

← Volver a inicio

Objetivo

Registrar y gestionar los pagos de servicios y refacciones, mediante un sistema de control financiero, para asegurar el seguimiento de ingresos, facturación y estados de pago dentro del negocio.

Usuarios del subsistema

  • Cliente
  • Personal administrativo
  • Administrador

Requerimientos por usuario

Requerimientos del Cliente

  • Consultar su historial de pagos.
    Campos: ID Cliente, ID Pago, ID Orden, Monto, Fecha de Pago, Estatus
  • Recibir notificaciones de pagos pendientes o realizados.
    Campos: ID Cliente, ID Pago, Estatus, Monto, Fecha de Pago

Requerimientos del Personal Administrativo

  • Registrar pagos.
    Campos: ID Pago, ID Cliente, ID Orden, Monto, Método de Pago, Fecha de Pago, Estatus, Notas
  • Actualizar estatus de pago.
    Campos: ID Pago, Estatus, Fecha de Pago, Notas
  • Generar recibos y reportes.
    Campos: ID Pago, ID Cliente, ID Orden, Monto, Método de Pago, Fecha de Pago, Estatus

Requerimientos del Administrador

  • Registrar y borrar pagos.
    Campos: ID Pago, ID Cliente, ID Orden, Monto, Método de Pago, Fecha de Pago, Estatus, Notas
  • Actualizar cualquier registro de pago.
    Campos: ID Pago + cualquier otro campo editable (Cliente, Orden, Monto, Método de Pago, Estatus, Fecha, Notas)
  • Leer información completa del subsistema.
    Campos: Todos los campos (Pagos, Clientes, Órdenes, Facturación, Estatus, Fechas, Notas)
